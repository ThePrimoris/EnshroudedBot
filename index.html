About

The Flame is a versatile Discord moderation bot designed to enhance server management within the Official Enshrouded Discord community. It provides a wide range of powerful tools and commands tailored to streamline moderation tasks, ensure server security, and improve user interaction.

Key Features:

  • Comprehensive user management functionalities:
    • Ban, mute, and warn users with customizable options.
  • Advanced message monitoring:
    • Monitor specified channels and configure actions based on content.
  • Server configuration options:
    • Manage roles, permissions, and server settings efficiently.

Empowering administrators with these tools, The Flame ensures a safe and welcoming environment for all community members in the Official Enshrouded Discord server.

Moderation Commands

Name Description Example
/mute Mutes a specified user for a specified duration /mute @user 10m
/mmute Mutes multiple users for a specified duration /mmute @user1 @user2 10m
/unmute Unmutes specified user /unmute @user
/munmute Unmutes multiple users /munmute @user1 @user2
/ban Bans a specified user with a reason /ban @user spamming
/tempban Temporarily bans a user for a specified duration /tempban @user 1d
/forceban Bans a user by user ID, even if they are not in the server /forceban userID
/mban Bans multiple users by their user ID, with a reason /mban userID1 userID2 spamming
/unban Unbans a user /unban @user
/munban Unbans multiple users /munban @user1 @user2
/info Provides information about a selected user /info @user
/warn Warns a specified user with a reason. DMs the user. /warn @user spamming
/note Adds a note on specified user. Does not DM them. /note @user needs improvement
/message Sends a message with the bot to a specified channel. /message #general Hello everyone!
/slowmode Changes slowmode duration in a specified channel. /slowmode #general 10s
/purge Deletes specified messages or messages from a specified user in a channel. /purge 10 or /purge @user
/kick Kicks a user from the server. /kick @user
